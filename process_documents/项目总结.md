## 🏗️ 技术架构

### 技术栈

- **语言**: Python 3.8+
- **PDF解析**: pdfplumber
- **向量数据库**: ChromaDB
- **AI模型**: OpenAI API (GPT-4/GPT-3.5)
- **框架**: LangChain (可选)

### 架构设计

```
数据层 (PDF教材)
    ↓
解析层 (pdf_parser.py)
    ↓
存储层 (ChromaDB向量数据库)
    ↓
知识层 (knowledge_base.py)
    ↓
AI层 (ai_agent.py + OpenAI)
    ↓
应用层 (main.py + 用户界面)
```

### 核心特性

1. **RAG架构** (Retrieval-Augmented Generation)
   - 检索相关教材内容
   - 增强AI回答准确性
   - 提供引用来源

2. **智能分块**
   - 按句子分割文本
   - 保留上下文完整性
   - 优化检索效果

3. **元数据管理**
   - 记录页码、科目、来源
   - 支持精确定位
   - 便于追溯验证

## 🚀 使用流程

### 首次安装

```bash
# 1. 运行自动化安装
./setup.sh  # macOS/Linux
setup.bat   # Windows

# 2. 配置API密钥
编辑 .env 文件

# 3. 测试系统
python test_system.py

# 4. 初始化知识库
python init_knowledge_base.py
```

### 日常使用

```bash
# 快速启动
./run.sh    # macOS/Linux
run.bat     # Windows

# 或手动启动
source venv/bin/activate
python main.py
```

## 💡 核心功能说明

### 1. PDF解析 (pdf_parser.py)

**功能:**
- 自动识别科目
- 提取文本内容
- 智能分块处理
- 保留元数据

**特点:**
- 支持中文PDF
- 处理多页文档
- 进度实时显示
- 错误处理完善

### 2. 知识库管理 (knowledge_base.py)

**功能:**
- 向量化存储
- 语义检索
- 统计分析
- 数据持久化

**特点:**
- 使用ChromaDB
- 支持批量操作
- 自动索引
- 快速查询

### 3. AI问答 (ai_agent.py)

**功能:**
- 智能问答
- 上下文检索
- 引用标注
- 交互界面

**特点:**
- RAG架构
- 专业提示词
- 多轮对话
- 友好界面

## ⚠️ 注意事项

1. **API密钥安全**
   - 不要将.env文件提交到版本控制
   - 定期更换API密钥
   - 注意API使用额度

2. **知识库维护**
   - 定期备份data目录
   - 教材更新后需重新初始化
   - 注意磁盘空间

3. **使用限制**
   - AI回答基于教材内容
   - 不能替代系统学习
   - 建议结合实际练习
